Este exercício é uma continuação do Laboratório 11c.

A sua tarefa é escrever um programa em Linguagem C (lab11d.c) que descubra se o robô é capaz de sair do labirinto ou se ele fica preso. Se ele consegue sair, seu programa deverá dizer qual o menor número de passos necessário para isso.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • A forma de entrada é idêntica ao do laboratório 11c
  • A entrada será formada pro diversas configurações de labirintos.
  • Cada labirinto será lido da seguinte maneira:
    • Uma linha com três números inteiros positivos N, L e C que significam:
      • N número de níveis do labirinto (1 ≤ N ≤ 10)
      • L número de linhas existentes nas matrizes que representam cada nível (1 ≤ L ≤ 10)
      • C número de colunas existentes nas matrizes que representam cada nível (1 ≤ C ≤ 10)
    • Uma seqüência de N matrizes L x C representando cada um dos níveis.
    • O ponto inicial será denotado pela letra I e aparecerá apenas uma vez no labirinto.
    • O ponto final será denotado pela letra F e aparecerá apenas uma vez no labirinto.
    • Áreas de trânsito livre são representadas por pontos (.).
    • As paredes são representadas por #.
  • A trinca 0 0 0 indica o final da entrada.
  • Para melhor entendimento da entrada, observe os exemplos.


Saída
  • Para cada labirinto lido da entrada, seu programa deverá descobrir se o robô consegue sair ou não do labirinto:
  • Se o robô consegue sair e P indica o número mínimo de passos que ele precisa dar, a seguinte mensagem deverá ser impressa:
    • Sai do labirinto em P passos
  • Se o robô não consegue sair, a seguinte mensagem deverá ser impressa:
    • Preso no labirinto
  • A trinca 0 0 0 indica o final da entrada e, por isso, nada deve ser impresso sobre ela.

Neste laboratório voltaremos ao popular passatempo conhecido como "Caça-Palavras", que foi abordado no Laboratório 07b.

Como vocês devem se lembrar, o problema pode ser definido da seguinte forma. Dada uma matriz de caracteres (letras ou números) e uma lista de palavras (letras e números são considerados caracteres válidos para palavras), determinar se as palavras podem ser encontradas na matriz. Uma palavra pode ser escrita em oito sentidos diferentes (N, NE, L, SE, S, SO, O, NO). Para cada palavra, deseja-se saber todas as posições iniciais onde ela pode ser encontrada (caso exista alguma).

Porém, nesse laboratório introduziremos duas pequenas alterações. Agora, uma letra pode ser utilizada várias vezes para formar uma palavra e várias direções podem ser utilizadas para uma mesma palavra.

Por exemplo, suponha a seguinte matriz:

1234TR
0CASRT
0XGTBK

Segundo a nova definição do problema, a palavra CASA pode ser encontrada na posição (1, 1) da matriz. A palavra começa no C, segue para a direita passando pelo A e pelo S e finaliza voltando para a esquerda, utilizando novamente o A. Como outro exemplo, a próxima matriz possui a palavra ARARAQUARA na posição (2, 4):

1f34aR1234TR
fRTTRU1g34tR
0XGTAQ92y4Tu
0XabBKh23jTR

Escreva um programa em linguagem C (lab11a.c) que resolve o problema proposto acima.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • A entrada poderá conter vários conjuntos de teste.
  • Cada conjunto de teste começará com uma trinca de inteiros positivos L, C e P tais que 0 ≤ L, C, P ≤ 50. Estes valores significam:
    • L: Número de linhas da matriz de caracteres
    • C: Número de colunas da matriz de caracteres
    • P: Número de palavras a serem procuradas na matriz
  • Após esta trinca de números, a entrada fornecerá a matriz de caracteres (L linhas e C Colunas).
  • A última informação de cada conjunto de teste é a lista de P palavras a serem procuradas na matriz. Cada palavra terá no mínimo 1 e no máximo 50 caracteres.
  • A última linha da entrada conterá a trinca 0 0 0 indicando o seu final.
  • A leitura deverá ser feita a partir da entrada padrão.


Saída
  • Em cada conjunto de teste, para cada umas das P palavras do conjunto, caso a palavra possa ser encontrada na matriz, deverá ser impressa a seguinte mensagem para cada uma das posições iniciais onde a palavra pode ser lida:
    • palavra: linha X, coluna Y
    Onde palavra é a palavra em questão, X é o número da linha (1 ≤ X ≤L) e Y é o número da coluna (1 ≤ Y ≤C).
  • Caso a palavra não possa ser localizada em nenhuma posição da matriz, a seguinte mensagem deve ser impressa:
    • palavra: nao encontrada
  • Ao final de cada conjunto de teste deve ser impressa a seguinte mensagem:
    • ------ FIM ------
  • Após esta mensagem, uma linha em branco deve ser impressa.
  • A trinca 0 0 0 indica o final da entrada e, por isso, nada deve ser impresso sobre ela.
  • A saída deverá ser escrita na saída padrão.
  • Observe os exemplos e repare na ordem de impressão quando uma palavra possuir mais de uma ocorrência.

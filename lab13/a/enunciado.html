A primeira edição do BBB - Big Brother Birigüi foi um sucesso de audiência. Os diretores da TV de Birigüi decidiram realizar uma nova edição do Reality Show.

Para essa nova edição, o diretor do programa decidiu abrir a participação a todos que quisessem. Dessa maneira, não existe mais um número limite de participantes.

O restante das regras continuam idênticas: um número inteiro M (M ≥ 1) define a quantidade de pessoas que devem ser percorridas na lista antes de uma ser eliminada. O processo de eliminação se repete enquanto houver mais de uma pessoa na lista.

Nesta tarefa você escreverá um programa em linguagem C (lab13a.c) que imprime a lista de eliminados do BBB II - Big Brother Birigüi II.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • O programa deverá receber uma lista de duplas de números inteiros positivos N ≥ 1 e M ≥ 1 (uma dupla por linha);
    • N: Indica o número de participantes do programa;
    • M: Indica o número utilizado no sorteio para percorrer a lista de participantes não eliminados.
  • A última linha da lista deverá ter a dupla 0 0 que indicará ao programa o fim da entrada e, portanto, o fim da execução;
  • A leitura deverá ser feita a partir da entrada padrão.


Saída
  • Todos os procedimentos de impressão da saída já estão implementados no esboço. Em caso de dúvidas, verifique os exemplos.

// Funcao que imprime os identificadores da lista ligada

void imprime(apLista atual, apLista inicio) {

  if (atual) {
    if (atual == inicio) {
      printf("(");

// Programa principal (PROIBIDO ALTERAR)
int main() {
  // declara as variaveis
  int pessoas, numero, i, eliminado, tamanho;
  apLista lista;

  // Le a entrada
  scanf("%d %d", &pessoas, &numero);

  // Processa se existir pelo menos uma pessoa
  while (pessoas > 0) {

    // No comeco a lista deve estar vazia
    inicializa(&lista);

    // Cria a lista circular de participantes
    for (i = 1; i <= pessoas; i++) {
      insere(&lista, i);
    }

    // Imprime a lista inicial de participantes
    printf("Lista inicial de participantes:\n");
    imprime(lista, lista);

    // Enquanto nao eliminou todos os candidatos
    while (!vazia(lista)) {
      movePonteiro(&lista, numero - 1);
      eliminado = retira(&lista);
      tamanho = conta(lista, lista);

      // Para criar suspense, nao anuncia a ordem
      // das duas ultimas eliminacoes
      if (tamanho > 1) {
        printf("\nParticipante eliminado = %d\n", eliminado);
        printf("%d participantes ainda no programa:\n",
               tamanho);
        imprime(lista, lista);
      }
    }

    // Anuncia o vencedor do programa
    printf("\n\nE o vencedor do BBB II eh...\n");
    printf("... participante numero %d!\n\n\n", eliminado);

    // Le uma nova entrada
    scanf("%d %d", &pessoas, &numero);
  }

  return 0;
}

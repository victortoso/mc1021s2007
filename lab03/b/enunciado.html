Um simpático caracol está na base de um poço de 600 centímetros de altura. Enquanto o sol está alto, ele consegue subir 300 centímetros mas durante a noite ele dorme e se descuida escorregando 100 centímetros para baixo. Além disso, conforme os dias vão passando o nosso amigo caracol começa a ficar cansado e consegue subir apenas 30 centímetros a menos que no dia anterior.

O problema aqui é descobrir em que dia o nosso amigo caracol irá conseguir sair do poço, ou seja, o primeiro dia em que a altura total subida será maior que a altura do poço. Um dia consiste em um período de sol (diurno) seguido por um período de escuridão (noturno).

Como podemos ver na tabela abaixo, o nosso amigo caracol consegue sair do poço no terceiro dia.

Dia Distância escalada Altura no final do período diurno Altura no final do período noturno
0 300 300 200
200 270 470 370
370 240 610 --

Neste laboratório você deverá escrever um programa em linguagem C (lab03b.c) que generalize o problema do caracol descrito acima.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.

Entrada

  • O programa deverá receber uma lista composta por 4 números por linha. Os números significarão, respectivamente:
    • A: Altura do poço;
    • S: Distância que o caracol consegue subir no primeiro dia;
    • D: Distância que o caracol desce durante a noite;
    • C: Fator de cansaço, ou seja, quanto a menos o caracol sobe dia após dia.
  • O caracol pode ficar muito cansado, mas ele consegue, de dia, pelo menos se manter na mesma posição e, portanto, nunca desce o poço durante o período diurno. Isso significa que ele nunca "sobe" uma distância negativa;
  • Como o caracol dorme durante a noite, ele não consegue impedir o escorregamento. Assim, durante a noite ele sempre desce D centímetros;
  • A última linha da lista deverá ter a seqüência 0 0 0 0 que indicará ao programa o fim da entrada e, portanto, o fim da execução;
  • A leitura deverá ser feita a partir da entrada padrão.
Saída
  • Para cada conjunto de 4 números A, S, D e C o programa deverá indicar o número do dia em que o caracol teve sucesso (conseguiu sair do poço) ou falhou (voltou para o fundo do poço);
  • Se o caracol teve sucesso no dia X, ou seja se a altura escalada for maior que a altura do poço, o programa deverá escrever:
    • Chegou ao topo no dia X
  • Se o caracol falhou no dia X, ou seja se a altura escalada se tornar negativa, o programa deverá escrever:
    • Voltou ao chao no dia X
  • O resultado de cada conjunto de 4 números A, S, D e C deverá ser impresso em uma linha;
  • A seqüência 0 0 0 0 indica o final da entrada e, portanto, nada deve ser impresso sobre ela;
  • Os resultados do programa deverão ser escritos na saída padrão.

#include <stdio.h>

int main() {
  int altura, sobe, desce, cansa, dia, atual;

  // Le a entrada
  scanf("%d %d %d %d", &altura, &sobe, &desce, &cansa);

  // Processa se a altura for maior que 0
  while (altura > 0) {

    // Inicializa dia e distancia atual


    // Calcula em quantos dias o caracol consegue
    // subir o muro ou enquantos dias ele desiste


    // Imprime a resposta
    if (atual > altura) {
      printf("Chegou ao topo no dia %d\n", dia);
    } else {
      printf("Voltou ao chao no dia %d\n", dia);
    }

    // Le uma nova entrada
    scanf("%d %d %d %d", &altura, &sobe, &desce, &cansa);
  }

  return 0;
}

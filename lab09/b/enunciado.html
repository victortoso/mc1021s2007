O ácido desoxirribonucléico (DNA) é composto por uma seqüência de pares de bases de nucleotídeos que formam uma estrutura de dupla hélice. Através de uma série de complexos processos bioquímicos a seqüência de DNA é traduzida em proteínas, que são fundamentais para os seres vivos.

Um DNA é composto por duas fitas complementares de nucleotídeos. Cada fita de DNA é formada pelas bases nucleotídicas Adenina (A), Citosina (C), Guanina (G) e Timina (T). Dada uma fita de DNA é possível construir a fita complementar trocando cada base pela sua complementar e invertendo a ordem das bases. Adenina e Timina são bases complementares, assim como Citosina e Guanina. As direções de cada fita são opostas. O pareamento das bases das fitas complementares é o que garante a estabilidade do DNA.

Tipicamente um DNA é representado apenas por uma de suas fitas, já que é fácil obter a sua fita complementar. Por exemplo, a seqüência CACTCGCTTGGAT representa uma fita de DNA cuja complementar é ATCCAAGCGAGTG. Veja como ambas as fitas podem ser pareadas de tal forma que sejam unidas pelas suas bases complementares:

A partir de uma fita de DNA, é produzido, num processo conhecido como transcrição, um ácido ribonucléico (RNA), conhecido como RNA mensageiro (ou mRNA). O RNA transcrito é idêntico a fita de DNA com exceção que a base Timina é substituída pela base Uracila (U). Por exemplo, a partir da fita CACTCGCTTGGAT é possível obter o RNA mensageiro CACUCGCUUGGAU.

É a seqüência de bases de um mRNA que determina a proteína que será construída. As bases de um mRNA podem ser vistas como uma seqüência de codons, sendo que cada codon é composto por exatamente três bases. No processo de tradução cada codon é associado a um aminácido de modo a construir uma proteína.

Existem 64 codons e apenas 20 aminoácidos. A relação entre os codons e os aminoácidos é conhecida como código genético. Toda proteína inicia-se pelo codon AUG, chamado de start codon, e termina em um dos três codons especiais UAA, UGA ou UAG, chamados de stop codons.

Uma protéina pode ser traduzida a partir de qualquer uma das duas fitas do DNA. Fitas diferentes podem gerar proteínas diferentes. Um trecho de DNA produz uma proteína válida apenas se a partir de um start codon é possível encontrar um stop codon, separados exatamente por um número de nucleotídeos múltiplo de 3, ou seja, entre o start e o stop codon devem existir apenas codons válidos. O codon AUG, além de servir como start codon, também é responsável pela tradução do aminoácido Metionina.

A sua tarefa será escrever um programa em Linguagem C (lab09b.c) que dado um trecho de uma seqüência de DNA (limitado a 200 nucleotídeos) determina se é possível obter a tradução de uma proteína. A tradução deve ser tentada no trecho fornecido e em seu complemento reverso. Note que computacionalmente a tradução de um DNA para uma proteína pode ser feita sem o passo intermediário da obtenção do RNA mensageiro. Para tanto basta adaptar a tabela acima para lidar com trincas de nucleotídios (A, C, G e T).

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • Neste laboratório, cada arquivo de teste será composto por diversas seqüências de DNA (limitadas em 200 nucleotídeos cada), uma por linha.
  • A última linha é composta apenas por um * indicando o fim da entrada.


Saída
  • Para cada seqüência de DNA fornecida como entrada, o programa deverá tentar traduzir uma proteína válida. Caso exista tal proteína, o programa deverá imprimí-la. Caso exista mais de uma proteína válida, seu programa pode imprimir qualquer uma delas. O start (representado pelo aminoácido M) e o stop codon (representado pelo caracter *) devem ser impressos antes e depois dos aminoácidos da proteína. Para este problema, também consideraremos como válida uma proteína composta apenas por um start e por um stop codon (neste caso o programa deverá imprimir apenas M*).
  • Caso não exista nenhuma tradução válida, em nenhuma das fitas, seu programa deverá imprimir a seguinte mensagem:
    • Impossivel traduzir

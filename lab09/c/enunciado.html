Miopia é o nome dado ao erro de refração da luz no olho, o que acarreta uma focalização da imagem antes de chegar à retina. Uma pessoa míope consegue ver objetos próximos com nitidez, mas os objetos distantes são visualizados como se estivessem embaçados (desfocados).

Um dos primeiros sintomas de um míope é a má visão de longe, estando a visão de perto preservada.

No entanto, é evidente que se um indivíduo tiver um alto grau de miopia, para ver bem, mesmo de perto, teria que aproximar-se muito dos objetos, o que é geralmente é muito cansativo e incômodo.

Mr. Quincy Magoo é um aposentado bem disposto e saudável. Ele possui um único problema de saúde (que ele se recusa a admitir): uma "pequena" miopia (13 graus em ambos os olhos).

Seu principal passatempo atualmente é visitar as comunidades dedicadas a ele no Orkut. Existem várias, sendo a mais popular delas com 2127 membros.

Os membros da comunidade participaram recentemente de um chat com Mr. Magoo e descobriram que apesar de contente com existência da comunidade ele considerava que o número que representa o tamanho da comunidade deveria ser exibido no site com maior orgulho (leia-se com dígitos grandes o suficiente para ele conseguir enxergar a uma distância razoável, digamos, a 5 cm da tela do computador).

Você como dono da comunidade "Mr. Magoo Rules", resolveu criar um programa, para ser usado na comunidade, que permita exibir números em ASCII de forma adequada para a visualização de Mr. Magoo.

A sua tarefa será escrever um programa em Linguagem C (lab09c.c) que dado um grau de miopia e um número qualquer (limitado a 100 dígitos) imprima na tela o número em um formato adequado para a leitura.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • Neste laboratório, cada arquivo de teste será composto por diversas configurações de entrada.
  • Cada configuração é composta por uma linha contendo um número inteiro positivo grau seguido por uma string número (limitada a 100 caracteres).
  • grau refere-se a miopia do usuário e número contém os dígitos que devem ser impressos no formato adequado ao grau de miopia.
  • A entrada termina quando grau for igual a zero.


Saída
  • Para cada configuração de entrada o programa deverá imprimir o número usando dígitos similares aos usados em dispositivos como calculadoras eletrônicas.
  • Cada dígito é composto por 7 traços sendo 3 deles horizontais e 4 verticais. Quando ligado, o traço deverá ser formado por tantos caracteres "#" quanto for o grau de miopia fornecido. Quando desligado o traço deverá ser representado de forma similar, usando caracteres ".".
  • A direita de cada dígito deve-se imprimir um espaço em branco. Ao final da impressão de cada número, uma linha em branco deve ser impressa.
  • Veja detalhes do formato de saída no esboço do programa e nos exemplos.

#include <stdio.h>
#include <string.h>


// Imprime o numero num formato adequado ao grau de miopia
void imprimeNumero(char numero[], int grau) {

}


// Programa principal (PROIBIDO ALTERAR)
int main() {
  int grau;
  char numero[101];
  
  // Le um grau de miopia
  scanf("%d", &grau);

  // Enquanto for uma entrada valida
  while (grau) {

    // Le um numero (sequencia de digitos)
    scanf("%s", numero);

    imprimeNumero(numero, grau);

    printf("\n");

    // Le um novo grau de miopia
    scanf("%d", &grau);
  }

  return 0;
}

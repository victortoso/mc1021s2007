Em muitos lugares, o jogo de cartas Guerra é muito popular entre as crianças. Como veremos a seguir, o resultado do jogo depende da distribuição inicial das cartas entre dois jogadores, A e B.

Este jogo utiliza 52 cartas de baralho e ignora os seus naipes. Apenas a numeração das cartas importa para o jogo que utiliza 4x13 cartas: Dois (2), Três (3), Quatro (4), Cinco (5), Seis (6), Sete (7), Oito (8), Nove (9), Dez (T), Valete (J), Rainha (Q), Rei (K) e Ás (A). Entre estas cartas, o Ás é a mais forte e o Dois é a mais fraca (2 < 3 < ... < 9 < T < J < Q < K < A).

No início do jogo, cada jogador segura um monte de cartas em suas mãos com as faces viradas para baixo. O objetivo do jogo é ganhar todas as cartas.

O jogo começa com uma Batalha. Ambos os jogadores mostram as cartas que estão nos topos de seus respecitvos montes. Aquele que tiver a maior carta pega as duas cartas e adiciona (com a face virada para baixo) no final de seu monte. É importante observar que a maior carta (carta vencedora) é adicionada primeiro. Na próxima rodada, os dois jogadores viram as cartas que estão nos topos de seus montes e tudo se repete.

Se as duas cartas que forem viradas pelos jogadores forem iguais (ou seja, se tiverem o mesmo número) a Batalha vira uma Guerra. As cartas empatadas permanecem na mesa e ambos os jogadores jogam suas próximas cartas. Caso as novas cartas sejam diferentes, aquele que tiver a maior carta pega a mesa para ele e adiciona todas as quatro cartas em seu monte. Se houver um novo empate, a guerra continua. A guerra continuará enquanto as cartas viradas forem iguais. Quando houver uma diferença, o jogador dono da maior carta adiciona a carta vencedora no final de seu monte, seguida pela carta perdedora e por todo o resto das cartas que estão na mesa, na ordem inversa de que elas foram jogadas.

Por exemplo, vamos assumir que os jogadores A e B possuem os seguintes montes (primeira carta, em azul, é o topo do monte ):

Finalmente, a guerra termina com o jogador A mostrando um Rei que é maior do que o Sete do jogador B. Note que, a maior carta sempre é colocada por cima da cartas que estão na mesa:

B QQJJTT99887KA
Mesa AAKK7K
Como a guerra terminou, o jogador vencedor (A) coloca as cartas da mesa em ordem inversa no final do seu monte e o jogo fica na seguinte situação:
Jogador Monte
A QQJJTT998877AK7KKAA
B QQJJTT99887KA
Mesa -

Observação importante: Note que durante a seqüência de eventos demonstrada acima, apenas uma Batalha ocorreu. A próxima rodada, daria início a uma nova Batalha.

O destino do jogo pode se encaminhar para 4 situações:

  1. Jogador A ganha o jogo quando o monte do jogador B acaba e A ainda tem cartas.
  2. Jogador B ganha o jogo na situação simétrica.
  3. Ocorre um empate, quando ambos os jogadores acabam suas cartas simultaneamente.
  4. O jogo é encerrado se for alcançado o número máximo de Batalhas pré-determinado. Neste caso, temos:
    1. Ganha o jogador que tiver o maior monte
    2. Se os montes forem de tamanhos iguais, teremos um empate

A sua tarefa será escrever um programa em Linguagem C (lab09a.c) que simule o jogo Guerra a partir da configuração inicial dos montes de cartas de dois jogadores.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • Neste laboratório, cada arquivo de teste será composto por diversas configurações de entrada.
  • Cada configuração é composta por uma linha contendo um número inteiro positivo N seguido por duas linhas contendo duas strings A e B representado os montes dos jogadores.
  • N indica o número máximo de batalhas.
  • A entrada termina quando N for igual a zero.


Saída
  • Para cada configuração de entrada o programa deverá imprimir o estado inicial e após cada uma das batalhas.
  • Veja detalhes do formato de saída no esboço do programa e nos exemplos.

#include <stdio.h>
#include <string.h>

#define tamanhoBaralho 52

// Define um monte de cartas (PROIBIDO ALTERAR)
typedef struct _monte {
  char cartas[tamanhoBaralho];
  int inicio, fim, quantidade;
} Monte;


// Inicializa um monte de cartas (PROIBIDO ALTERAR)
void inicializaMonte(Monte * monte, char cartas[]) {

  strcpy((*monte).cartas, cartas);
  (*monte).inicio = 0;
  (*monte).quantidade = strlen(cartas);
  (*monte).fim = (*monte).quantidade;
}


// Verifica se um monte de carta esta vazio
int vazioMonte(Monte * monte) {

  return (*monte).quantidade == 0;
}


// Retorna o tamanho de um monte
int tamanho(Monte * monte) {
  
  return (*monte).quantidade;
}


// Verifica o valor de uma carta (PROIBIDO ALTERAR)
int valor(char carta) {
  
  switch (carta) {
    case 'T': return 10; 
    case 'J': return 11;
    case 'Q': return 12;
    case 'K': return 13;   
    case 'A': return 14;   
    default: return carta - '0';
  }
}


// Disputa uma guerra
void guerra(Monte * monte1, Monte * monte2) {


}


// Imprime um monte de cartas (PROIBIDO ALTERAR)
void imprime(Monte * monte, int jogador) {
  int i;

  printf("Jogador %d: ", jogador);

  for (i = (*monte).inicio; i != (*monte).fim; 
       i = (i + 1) % tamanhoBaralho) {
    printf("%c", (*monte).cartas[i]);
  } 
  printf("\n");
}


// Programa principal (PROIBIDO ALTERAR)
int main() {
  char cartas1[tamanhoBaralho+1], cartas2[tamanhoBaralho+1];
  Monte monte1, monte2;
  int batalha, maxBatalhas;

  // Le o numero maximo de batalhas
  scanf("%d", &maxBatalhas);

  while (maxBatalhas) {

    // Le os montes de cartas dos dois jogadores
    scanf("%s %s", cartas1, cartas2);
    
    // Inicializacoes
    inicializaMonte(&monte1, cartas1);
    inicializaMonte(&monte2, cartas2);
    batalha = 0;

    printf("No inicio:\n");
    imprime(&monte1,1);
    imprime(&monte2,2);

    // Enquanto nao acabou o jogo
    do {
      batalha ++;

      guerra(&monte1, &monte2);

      printf("\nApos a %da rodada:\n", batalha);
      imprime(&monte1,1);
      imprime(&monte2,2);

    } while (batalha < maxBatalhas && 
	     !(vazioMonte(&monte1) || vazioMonte(&monte2)));

    // Imprime a resposta
    if (tamanho(&monte1) == tamanho(&monte2)) {
      printf("\nEmpate\n\n");
    } else if (tamanho(&monte1) > tamanho(&monte2)) {
      printf("\nJogador 1 venceu\n\n");
    } else {
      printf("\nJogador 2 venceu\n\n");
    }

    // Le o novo numero maximo de batalhas
    scanf("%d", &maxBatalhas);
  }

  return 0;
}

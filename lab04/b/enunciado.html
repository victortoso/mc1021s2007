A i-ésima estatística de ordem de um conjuto de n elementos é o i-ésimo menor elemento. O mínimo é a primeira estatística de ordem, e o máximo é a n-ésima estatística de ordem. Mediana é o ponto médio de um conjunto de valores. Quando n é ímpar e m = (n + 1)/2, a mediana é a m-ésima estatística de ordem. Quando n é par, existem duas medianas, a mediana inferior, que é a i-ésima estatística de ordem (i = n/2), e a mediana superior, que é a j-ésima estatística de ordem (j = (n/2) + 1). Neste contexto, conjunto é uma lista de número não necessariamente distintos ou ordenados.

Sua tarefa é escrever um programa em linguagem C (lab04b.c) que, dado o tamanho de um conjunto N (1 ≤ N ≤ 100), um inteiro positivo K (1 ≤ K ≤ N) e a lista de elementos do conjunto, determina a K-ésima estatística de ordem do conjunto.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • O programa deverá receber um ou mais conjuntos de testes (um conjunto por linha). Em cada linha, os dois primeiros valores serão, respectivamente, o tamanho do conjunto N e o a estatística de ordem desejada K. A seguir serão listados os N números (todos inteiros) que compõe o conjunto.
  • A última linha da entrada deverá conter um único número 0 que indicará ao programa o fim da entrada e, portanto, o fim da execução;
  • A leitura deverá ser feita a partir da entrada padrão.


Saída
  • Para cada conjunto de teste, o programa deverá imprimir a estatística de ordem desejada, no seguinte padrão:
    • 15a estatistica de ordem: 97
  • A linha 0 indica o final da entrada e, portanto, nada deve ser impresso sobre ela;
  • Os resultados do programa deverão ser escritos na saída padrão.

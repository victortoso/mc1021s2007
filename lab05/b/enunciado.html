Um pequeno robô foi programado para seguir as intruções que ele encontra em seu caminho.

A área onde o robô caminha é um tabuleiro que contém símbolos representando os quatro pontos cardeais (N - Norte, S - Sul, E - Leste e W - Oeste).

O robô, quando colocado em uma determinada posição do tabuleiro, lê a instrução contida nela e a executa. As possíveis instruções são:

  • N anda uma casa para cima
  • S anda uma casa para baixo
  • E anda uma casa para a direita
  • W anda uma casa para a esquerda

O robõ executa as instruções até conseguir sair do tabuleiro. Eventualmente, conforme a posição inicial e a configuração do tabuleiro, ele pode entrar em loop. A figura abaixo demonstra as duas situações. Note que a posição (0,0) é a do canto superior esquerdo do tabuleiro.

Sua tarefa é escrever um programa em linguagem C (lab05b.c) que, dados o tabuleiro de instruções e a posição inicial do robô, determine o número de passos que o robô dá até sair do tabuleiro ou que indique o número de passos que o robô levou para entrar em loop e o tamanho do loop.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • A entrada poderá conter vários mapas de instruções.
  • A entrada será composta por uma seqüências de 4 números inteiros M, N, X e Y tais que 0 ≤ M, N ≤ 100 e 0 ≤ X, Y < 100. Estes valores significam:
    • M Número de linhas do tabuleiro de instruções
    • N Número de colunas do tabuleiro de instruções
    • X Índice da linha onde o robô iniciará o percurso
    • Y Índice da coluna onde o robô iniciará o percurso
  • Após está seqüência de quatro números, a entrada terá M linhas contendo N símbolos representado os pontos cardeais.
  • A última linha da entrada conterá a seqüência 0 0 0 0 indicando o seu final.
  • A leitura deverá ser feita a partir da entrada padrão.


Saída
  • O programa deverá calcular o número de passos dado pelo robô até sair do tabuleiro ou até entrar em loop.
  • Seja A o número de passos que o robô gastou para sair do tabuleiro. A saída deverá ser impressa no formato:
    • Robo saiu do tabuleiro apos A passo(s)
  • Sejam B o número de passos que o robô gastou para entrar em loop e C o tamanho do loop. A saída deverá ser impressa no formato:
    • Robo entrou em loop apos B passos (loop de tamanho C)
  • A seqüência 0 0 0 0 indica o final da entrada e, por isso, nada deve ser impresso sobre ela.
  • A saída deverá ser escrita na saída padrão.


Exemplos
Entrada Saída

N E E S W E

Robo entrou em loop apos 10 passos (loop de tamanho 8)


Esboço
#include <stdio.h>

int main() {
  int linhas, colunas, x, y, i, j;
  char grid[100][100];

  // Le a entrada
  scanf("%d %d %d %d", &linhas, &colunas, &x, &y);

  // Enquanto a entrada for valida
  while ((linhas > 0) && (colunas > 0)) {

    // Le o grid onde o robo sera colocado
    for (i = 0; i < linhas; i++) {
      for (j = 0; j < colunas; j++) {
        scanf(" %c", &grid[i][j]);
      }
    }




    // Le uma nova entrada
    scanf("%d %d %d %d", &linhas, &colunas, &x, &y);
  }

  return 0;
}

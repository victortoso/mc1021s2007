Laboratório Nº 02a - Identificação de Números Primos

Peso: 3

Docente: Zanoni Dias

Escreva um programa em linguagem C (lab02a.c) que indentifique em uma lista de números inteiros maiores ou iguais a 2 aqueles que são primos.
Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.

Entrada
  • O programa deverá receber uma lista de números inteiros positivos (um número por linha) menores que 1000000;
  • O último elemento da lista deverá ser um número inteiro menor ou igual a 1 que indicará ao programa o fim da entrada e, portanto, o fim da execução;
  • A leitura deverá ser feita a partir da entrada padrão.
Saída
  • Seja N um número pertencente à lista de entrada:
    • Se N não é primo, o programa deve imprimir: N is not prime
    • Se N é primo, o programa deve imprimir: N is prime
  • O resultado de cada número deverá ser impresso em uma linha;
  • O número inteiro menor ou igual a 1 presente na lista indica o final da entrada e, portanto, nada deve ser impresso sobre ele;
  • Os resultados do programa deverão ser escritos na saída padrão.

Esboço
// Incluir cabecalho padrao
#include <stdio.h>

int main() {
  // Se necessario, declarar outras variaveis inteiras
  int n, primo;

  // Le a entrada
  scanf("%d", &n);

  // Processa se o numero for maior que 1
  while(n > 1) {

    // A principio todo numero eh primo
    primo = 1;

    // Incluir aqui um teste de primalidade.
    // Se n nao for primo,fazer:
    // primo = 0;

    // Imprime a resposta
    if (primo) {
      printf("%d is prime\n", n);
    } else {
      printf("%d is not prime\n", n);
    }

    // Le uma nova entrada
    scanf("%d", &n);
  }

  return 0;
}

O tabuleiro e as peças de Xadrez são freqüentemente utilizados para a criação de novos quebra-cabeças. Um problema muito famoso é o das Oito Rainhas que consiste em descobrir uma maneira de dispor 8 rainhas em um tabuleiro 8 x 8 de modo que elas não possam se atacar.

Neste laboratório, iremos implementar um programa que solucione outro problema envolvendo peças de Xadrez.

Trata-se do problema do Hipismo que consiste em movimentar um cavalo em um tabuleiro de modo que ele passe por todos os obstáculos existentes na ordem definida. O tabuleiro 5 x 5 abaixo mostra os 8 possíveis movimentos que um cavalo pode executar. O C vermelho indica o local onde o cavalo se encontra e os Xs azuis mostram as posições para onde ele pode "pular".

  X   X  
X       X
    C    
X       X
  X   X  

Porém, para resolver o problema, além de cumprir o percurso definido, você deverá agir como Rodrigo Pessoa e sua égua Baloubet du Rouet e tentar fazer o percurso de maneira ótima (e, de preferência, sem refugar). Isso significa que você deverá fazer o percurso no menor número de passos possível.

Para isso, o programa receberá a dimensão do do tabuleiro em linhas e colunas e uma lista de obstáculos que definem o percurso a ser cumprido. O cavalo sempre começa no primeiro obstáculo e deverá chegar no último, passando por todos os outros.

Os tabuleiros abaixo mostram um exemplo de resolução do problema. O tabuleiro da esquerda é uma matriz de tamanho 6 x 6 que possui 4 obstáculos que devem ser percorridos (o1, o2, o3 e o4). O tabuleiro da direita enumera os passos que o cavalo deve executar para passar por todos os obstáculos.

A sua tarefa será escrever um programa em Linguagem C (lab12b.c) para resolver o problema proposto acima.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • A entrada será composta por diversas configurações de tabuleiros.
  • Cada configuração será formatada da seguinte maneira:
    • Uma trinca de valores inteiros positivos L, C e O (4 ≤ L, C≤ 20 e 2 ≤ O ≤ 20) com os seguintes significados:
      • L número de linhas do tabuleiro.
      • C número de colunas do tabuleiro.
      • O número de obstáculos contidos no tabuleiro.
    • Uma lista composta por O duplas de números inteiros X e Y (0 ≤ X < L, 0 ≤ Y < C) representado as posições onde os obstáculos se encontram.
  • A última linha da entrada é composta pela trinca 0 0 0 que indica o seu final.


Saída
  • Para cada configuração de tabuleiro lida da entrada, seu programa deverá calcular o número mínimo de passos D e imprimir a seguinte mensagem (uma linha para cada configuração):
    • Distancia total = D
  • A trinca 0 0 0 indica o final da entrada e nada deve ser impresso sobre ela.

Uma maneira de se representar um retângulo é listando as coordenadas de seus quatro vértices. Contudo, se o retângulo tiver seus lados paralelos aos eixos x e y, a sua representação pode ser feita listando apenas as coordenadas de seus vértices superior esquerdo e inferior direito.

Nesta tarefa você deverá escrever um programa em linguagem C (lab12e.c) que receberá uma lista de retângulos, representados pelos seus vértices superior esquerdo e inferior direito, e calculará a área da união de todos eles. Note que, para isso, você deve lidar com as intersecções existentes entre os retângulos. A figura abaixo mostra como é o cálculo da área da união de dois retângulos que possuem intersecção não nula.

Utilize o esboço do programa e lembre-se de colocar o cabeçalho e incluir comentários pertinentes.



Entrada
  • A entrada será composta por uma lista de conjuntos de retângulos representados pelos vértices superior esquerdo e inferior direito, sendo que cada conjunto será representado da seguinte forma:
    N
    XS1 YS1 XI1 YI1
    XS2 YS2 XI2 YI2
    ... ... ... ...
    XSN YSN XIN YIN
  • Nesta representação de conjunto de retângulos:
    • N é um inteiro que indica o número de retângulos que compõem o conjunto. Note que não há nenhum limite quanto ao valor máximo de N.
    • XSi e YSi representam as coordenadas do vértice superior esquerdo Si do retângulo i (1 ≤ i ≤ N)
    • XIi e YIi representam as coordenadas do vértice inferior direito Ii do retângulo i (1 ≤ i ≤ N)
  • A entrada termina quando for lido N igual a zero.
  • A leitura deverá ser feita a partir da entrada padrão.


Saída
  • Para cada conjunto de retângulos lido da entrada, a área A (da união de todos os retângulos) calculada deverá ser impressa no seguinte formato:
    • Area = A
  • Note que a área deve ter precisão de duas casas.
  • A saída deverá ser escrita na saída padrão.

#include <stdio.h>
#include <stdlib.h>

// Define um ponto (PROIBIDO ALTERAR)
typedef struct _ponto {
  double x;
  double y;
} Ponto;

// Define um retangulo (PROIBIDO ALTERAR)
typedef struct _retangulo {
  Ponto superior;
  Ponto inferior;
} Retangulo;

// Calcula o minimo de dois valores (PROIBIDO ALTERAR)
double min(double x, double y) {
  if (x < y) {
    return x;
  } else {
    return y;
  }
}

// Calcula o maximo de dois valores (PROIBIDO ALTERAR)
double max(double x, double y) {
  if (x > y) {
    return x;
  } else {
    return y;
  }
}

// Calcula a interseccao de dois retangulos (PROIBIDO ALTERAR)
Retangulo intersection(Retangulo a, Retangulo b) {
  Retangulo c;

  c.superior.x = max(a.superior.x, b.superior.x);
  c.superior.y = min(a.superior.y, b.superior.y);
  c.inferior.x = min(a.inferior.x, b.inferior.x);
  c.inferior.y = max(a.inferior.y, b.inferior.y);

  c.superior.x = min(c.superior.x, c.inferior.x);
  c.superior.y = max(c.superior.y, c.inferior.y);

  return c;
}

// Calcula a area de um retangulo (PROIBIDO ALTERAR)
double area(Retangulo r) {

  double altura = r.superior.y - r.inferior.y;
  double largura = r.inferior.x - r.superior.x;

  return altura * largura;
}

  }

  return 0;
}
